# 🧩 Proyecto Práctico: Lenguaje de Programación I, II y III


## 📌 Descripción

Este repositorio contiene una colección de **ejercicios prácticos** diseñados para apoyar a los estudiantes de las asignaturas **Lenguaje de Programación I, II y III**. 

Los problemas están organizados de forma progresiva, siguiendo la planificación educativa oficial, y abarcan desde los fundamentos de la programación hasta temas avanzados como estructuras de datos y programación orientada a objetos.

> 🎯 **Finalidad**: Servir como material de apoyo para clases, laboratorios y estudio autónomo.

## 📚 Estructura por cursos

### 🔹 Lenguaje de Programación I (Fundamentos)
- Tipos de datos, variables y constantes
- Entrada / salida (cin, cout)
- Estructuras de control: condicionales (`if`, `switch`) y bucles (`for`, `while`, `do-while`)
- Funciones (parámetros por valor y referencia)
- Arreglos unidimensionales y cadenas de caracteres

### 🔸 Lenguaje de Programación II (Programación estructurada y avanzada)
- Arreglos bidimensionales (matrices)
- Punteros y gestión dinámica de memoria (`new`, `delete`)
- Estructuras (`struct`) y uniones (`union`)
- Manejo de archivos de texto y binario
- Recursividad
- Modularización y bibliotecas propias

### 🔹 Lenguaje de Programación III (Programación Orientada a Objetos y temas complementarios)
- Clases y objetos (encapsulación, constructores/destructores)
- Herencia y polimorfismo
- Sobrecarga de operadores
- Plantillas (funciones y clases)
- Excepciones (`try`, `catch`, `throw`)
- Introducción a la STL (vectores, listas, mapas)
